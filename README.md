Turing-Machine-Simulator
图灵机可视化模拟器
纯前端实现、零依赖浏览器端图灵机模拟器，支持规则自定义、经典模板一键加载、运行轨迹可视化与图像导出，适合计算机理论教学、算法学习与图灵机原理实践。
📌 项目简介
使用 HTML + CSS + 原生JavaScript 开发，无需后端、无需编译，直接打开 HTML 文件即可运行。
完美复刻标准图灵机运行逻辑，可视化展示纸带、读写头、状态转移全过程，内置多款经典图灵机案例。
✨ 核心功能
🎮 可视化纸带模拟，固定读写头 + 滑动纸带动态效果
📊 转移矩阵可视化编辑，直观修改运行规则
➕ 自由新增 / 删除运行状态，自定义状态命名
📚 内置多款经典图灵机模板，一键导入
⏯️ 播放 / 暂停 / 一键重置 全流程控制
⚡ 运行速度无级调速，快慢自由切换
📈 Canvas 实时记录完整运行历史轨迹
💾 浏览器本地持久化存储自定义规则，刷新不丢失
🖼️ 一键导出运行轨迹为 PNG 图片
🎯 当前执行规则自动高亮，便于调试
🖥️ 界面结构
顶部控制栏
运行控制按钮、速度调节滑块、实时状态、运行步数统计
左侧规则面板
状态转移编辑表格、状态增删、经典模板库
右侧可视化面板
实时纸带视图、读写头状态显示、全局运行轨迹画布
🚀 快速使用
1. 本地运行
新建文件夹，将代码保存为 index.html
使用任意现代浏览器（Chrome / Edge / Firefox）直接打开该文件
无需搭建环境、无需安装依赖，开箱即用
2. 基础操作
播放：启动图灵机自动单步运行
暂停：暂停当前运行流程
重置：清空纸带、复位读写头、清零步数
速度滑块：向左变慢、向右变快
导出图像：保存运行全过程轨迹图
3. 编辑自定义规则
规则格式：当前状态 + 当前读取符号 → 写入符号 + 移动方向 + 下一状态
点击 + 状态 新增运行状态
点击状态旁 × 删除多余状态
表格内直接修改：
写：写入纸带符号 0 / 1
移：左移L / 右移R
进：填写下一状态，填写 H 代表停机
修改自动本地保存，永久生效
4. 加载经典模板
左侧常用模板库点击按钮一键加载预设规则：
表格
模板名称	功能描述
BB1~BB4 忙碌海狸	经典忙碌海狸问题，停机前生成最多连续 1
二进制计数器	实现二进制连续自增计数
二进制加一	对纸带二进制数字执行 +1 运算
0-1 翻转填充器	遍历纸带翻转所有 0 和 1
右移复制	实现纸带符号向右复制
交替填充 0101	自动生成 0101 循环序列
向左搜索 1	向左遍历，匹配到 1 后停机
🧠 图灵机运行规则
纸带默认为无限长，空白位置默认值为 0
仅支持纸带符号：0、1
读写头仅可左右单向移动
遇到停机状态 H 或无匹配规则时，机器自动停止运行
运行历史横向为执行步数，纵向为纸带位置
🛠️ 技术栈
页面结构：HTML5
样式布局：CSS3 + CSS 变量 + Flex 布局
交互逻辑：原生 JavaScript
轨迹绘制：HTML5 Canvas
数据存储：localStorage 本地持久化
💾 本地存储说明
所有手动修改的状态、转移规则都会自动存入浏览器本地缓存，
关闭页面、刷新页面、重启浏览器配置均不会丢失，
如需清空配置，清除浏览器网站本地存储即可。
⚠️ 注意事项
尽量使用新版现代浏览器运行，不兼容老旧 IE 浏览器
不设置停机状态 H 会造成机器无限循环运行
状态建议使用大写英文字母命名，规范易读
轨迹画布会自动扩容，长时间运行可导出图片保存结果
📝 自定义示例
简易 01 翻转机
仅保留单个状态 A
设置规则：
A 读取 0 → 写入 1、右移、回到 A
A 读取 1 → 写入 0、右移、回到 A
点击播放即可实现无限向右翻转纸带符号
